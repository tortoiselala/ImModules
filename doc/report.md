### 功能需求
系统根据业务需求大致分为通讯服务、权限认证服务
通讯服务大致分为通讯模块，通讯录模块，动态通知模块以及个人中心模块
#### 通讯模块
通讯模块主要负责即时通信信息的发送和接收，可发送包括文字、表情、语言、图片在内的多种形式的内容

## 总结
这个项目主要是用于应付面试的，希望能够用到一些比较前沿的框架以及技术
从而能够在面试时有聊的东西。

目前主要用到的技术如下：

- Spring
- Spring MVC
- MyBatis
- MySQL
- Oauth协议
- JWT

之后引入NoSQL（Redis）引入Redis的目的在于两个方面

- token的存储，多客户端登录管理
- 热点数据缓存
- 消息记录临时存储（替代MQ）
